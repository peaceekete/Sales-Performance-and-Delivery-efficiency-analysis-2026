# Sales-Performance-and-Delivery-efficiency-analysis-2026
Sales performance analysis with a focus on profit drivers and delivery delays.

## Overview
This project analyzes sales and operational data to identify key revenue drivers, profitability trends, and delivery inefficiencies affecting business performance.

## Key Insights
- Step is the top-performing product with the highest revenue
- Electronics generates the highest profit across categories
- Top 5 products contribute significantly to overall revenue
- Average delivery time is 122 days, indicating major inefficiencies

## Tools Used
- Power BI
- Excel

## Features
- Interactive dashboard
- Revenue and profit analysis
- Category performance breakdown
- Delivery trend analysis

## Dataset
Sample sales dataset including product, category, revenue, profit, and delivery data.
